# 💀 逆天注册系统

> 一个让你怀疑人生的注册页面 —— 从填表到滴血，从灵魂契约到按钮追逐，集齐所有反人类验证交互。

---

## 📌 项目简介

这是一个**纯恶搞**、**行为艺术**级别的注册表单。表面是“注册会员账号”，实际上内嵌了 **40+ 层折磨验证**，包含但不限于：

- 强制选“其他”性别
- 滑块验证（会摇晃位移）
- 灵魂卖身契
- 原神 & 蔚蓝档案调研陷阱
- 20+ 道离谱问答题（声纹、色盲、记忆力、暗号、质数……）
- **滴血三重验证**（模拟外置血检设备，6步走完）
- 注册按钮追逐鼠标
- 投降旗 & 最终卖身契确认
- 全程随机骚扰（闪屏、输入篡改、蓝屏模拟、广告弹窗、步骤回退……）

⚠️ **警告**：本页面纯属娱乐，切勿用于任何真实注册场景。开发者不对任何因尝试注册而导致的血压升高、摔鼠标、砸键盘、信仰崩塌负责。

## ✨ 完整流程（从折磨到“解脱”）

### 1️⃣ 基础表单（看似正常，实则暗坑）
- **用户名**：4-30位，黑名单含 `admin`、`test`、常见英文名等，且随机判定不合格
- **密码**：≥8位，含大小写+数字+特殊字符，随机降级
- **手机号**：屏蔽常见号段，豹子号触发“天亮了”彩蛋
- **性别**：只能选“其他”，选“男/女”报错“该性别已被占用”，强制取消勾选

### 2️⃣ 滑块验证（会跳舞）
- 拖动滑块至最右端，但弹窗**横向移动 + 上下摇晃**，增加操作难度
- 验证通过后可能**随机重置**，让你再拖一次

### 3️⃣ 华为设备检测（User-Agent）
- 检测到华为/鸿蒙设备，弹出“华为好，华为美”洗脑文案

### 4️⃣ 用户调研（原神 & 蔚蓝档案）
- 必须选“是”，选“否”会被嘲讽，选“是”也会被骂

### 5️⃣ 灵魂绑定契约
- 条款离谱：灵魂永久绑定、放弃梦境数据、每日999封邮件、注销需公证……
- **必须勾选同意**才能继续

### 6️⃣ 活体检测（模拟人脸识别）
- 点击按钮，进度条走满即通过（无摄像头，纯纯模拟）

### 7️⃣ 双因素认证（2FA）
- 设置6位数字密钥 → 再输入一遍验证（密钥就是显示的数字）

### 8️⃣ 邮箱验证
- 固定验证码 `888888`

### 9️⃣ 安全提问
- 1 + 1 = ? 答对答错都能继续（答错被嘲讽，但照样放行）

### 🔟 系统审核
- 进度条加载，模拟审核过程，可能随机失败回退

## 💉 滴血三重验证（新增终极折磨）

审核通过后，**强制进行滴血验证**（模拟外置血检设备）：

- 6步流程：**设备连接 → 设备自检 → 血样采集 → 样本纯化 → 生物分析 → 身份确认**
- 设备屏幕实时显示状态（INIT → READY → CHK → SMPL → PURE → SEP → ANLYZ → MATCH → PASS）
- LED灯变色（绿/黄/红），检测槽闪烁
- 进度条动态填充，步骤指示器高亮
- 验证通过后显示成功对勾，设备边框亮绿

**整个过程耗时约 8.6 秒，无法跳过，且期间不能关闭弹窗。**

---

## 🏃 按钮追逐 & 投降旗

滴血验证通过后，**注册按钮开始追逐鼠标**，永远点不到。

右下角出现 **🏳️ 投降旗**，点击后按钮停止追逐，弹出最终确认框。

---

## 📝 最终确认（卖身契）

必须逐字输入：

```
我自愿放弃所有权利并永远忠诚于本平台
```

输入正确后，点击“确认注册”，才算真正完成（并显示成功 Toast）。

## 🎲 随机整蛊（持续骚扰）

页面运行期间会随机触发以下效果，**贯穿整个注册流程**：

- 背景闪烁（多色）
- 输入框内容被自动篡改（追加字符）
- 广告弹窗（虚假 VIP 领取）
- 蓝屏死机模拟（3秒后消失）
- 卡片抖动
- 文字翻转
- 输入框疯狂边框
- 随机 Toast 提示（“你被监视了”、“快跑！”）
- 按钮随机瞬移（未进入追逐模式时）
- 滑块验证随机重置
- 步骤随机跳跃/回退
- 审核随机失败（从头再来）
- 最终确认前 20% 概率触发“终极骗局”（重新加载页面）
